┌─────────────────────────────────────────────────────────────────┐
│ 🎨 HOW IT WORKS                                                 │
└─────────────────────────────────────────────────────────────────┘

  User Speaks
      ↓
  Browser Records Audio (MediaRecorder)
      ↓
  API Transcribes (Whisper - supports Hindi/English/Hinglish)
      ↓
  GPT Extracts Intent & Entities
      ↓
  Routes to Correct Page (flights/hotels/packages)
      ↓
  Auto-fills Form & Triggers Search
      ↓
  Speaks Result Back to User ✨

┌─────────────────────────────────────────────────────────────────┐
│ 🐛 TROUBLESHOOTING                                              │
└─────────────────────────────────────────────────────────────────┘

  "Mic not working"
  → Grant browser permission (click lock icon in URL bar)
  → Use HTTPS or localhost
  → Try different browser (Chrome/Edge recommended)

  "Voice processing failed"
  → Check API key in .env.local
  → Verify OpenAI account has credits
  → System will fallback to mock mode automatically

  "Search not triggered"
  → Check browser console for errors
  → Be more specific: "Find flights from X to Y on [date]"
  → Try clicking "Yes, search" button if shown
